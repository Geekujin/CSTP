# Useful Links

This file contains links to useful tools and resourcesthat may be beneficial during the degree.

## Data Sets
- [Malware Traffic Analysis](https://malware-traffic-analysis.net/) - Wireshark .pcap files containing malicious activity and IoCs
- [Virus Share](https://virusshare.com/) - A repository of malware samples
- [Awesome Cybersecurity Datasets](https://github.com/shramos/Awesome-Cybersecurity-Datasets) - A massive collection of various Cyber related data sets
- [Kaggle](https://www.kaggle.com) - Another great resource for data sets.

***

## Web Services & Search Engines
- [Any Run](https://app.any.run) - Online malware analysis sandbox
- [URLScan.io](https://urlscan.io) - URL scanner
- [Shodan](https://shodan.io) - A search engine for internet connected devices (a free full membership when registering with student email!)
- [CRT.Sh](https://crt.sh) - Search for all past and present SSL certs tied to a domain name
- [VirusTotal](https://virustotal.com) - Find information about malicious urls or files
- [Who.Is](https://who.is) - Domain registration and DNS information
- [WTF Is My IP?](https://wtfismyip.com/) - Displays your external IP address using NSFW language :)
- [CVE Details](https://www.cvedetails.com/) - A search engine for CVEs. Search by CVE ID, vendor or product.
